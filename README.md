## What is Ona?

**Ona** is a **multi-agent orchestration platform**. You talk to **Solin** — your personal agent — and Solin coordinates a **squad of specialists** (Writer, Researcher, Developer, Marketer, etc.) to get work done. It runs **locally or on a VPS** on **any machine**: Linux, macOS, Windows. Any CPU (x86_64, ARM64), any RAM tier, with or without GPU. Self-hosted, no vendor lock-in; you can use local models (Ollama, llama.cpp) or cloud APIs. Channels include web chat, Telegram, WhatsApp, Signal, and more. **Alpha** — we’re still in active development; install files and a Discord server are coming soon.

This repo holds **community-facing content** only: how to get in touch, contributor/maintainer path, and policies. The **main Ona app and code** live in a separate repository.

## Core vs community

- **Core (main Ona repo):** App code, boot, systems, production scripts, core docs. Single source of truth for the app.
- **Community (this repo):** Policies, maintainer/contributor onboarding, code of conduct, security reporting for the community, and links back to core.

For **code and core docs**, contribute to the main Ona repository and follow its CONTRIBUTING and style guide.
